# Procesamiento de Alto Volumen de Datos

Repositorio creado para organizar los trabajos, laboratorios, talleres y proyectos desarrollados en la clase de **Procesamiento de Alto Volumen de Datos**.

El contenido del repositorio está enfocado en el uso de herramientas para el análisis, procesamiento y transformación de datos, especialmente mediante **Python**, **PySpark**, **Apache Spark**, manejo de datasets, análisis exploratorio y construcción de modelos básicos de machine learning.

## Contenido del repositorio

El repositorio se encuentra organizado por carpetas según los temas y actividades trabajadas durante el curso.

### Configuración de Servidor Spark

Carpeta dedicada a la configuración inicial de un entorno de trabajo con **Apache Spark** y **PySpark**. Incluye ejercicios introductorios sobre creación de sesiones Spark, lectura de archivos CSV, uso de DataFrames y primeras transformaciones sobre datos estructurados.

### Taller Python

Contiene ejercicios prácticos de programación en **Python**, orientados al manejo básico de datos, estructuras, funciones y uso de librerías comunes para análisis.

### Taller Métricas de Rendimiento

Incluye actividades relacionadas con la evaluación de modelos y análisis de métricas. Se trabajan conceptos como accuracy, precision, recall, F1-score, matrices de confusión y otras medidas útiles para interpretar resultados.

### Parcial

Carpeta destinada a desarrollos, ejercicios o proyectos asociados a evaluaciones parciales del curso. Incluye análisis de datasets y aplicación de técnicas vistas en clase.

## Herramientas utilizadas

- Python
- Jupyter Notebook
- Apache Spark
- PySpark
- Hadoop HDFS
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Temas trabajados

- Configuración de entornos Spark.
- Lectura y procesamiento de archivos CSV.
- Manejo de DataFrames en PySpark.
- Limpieza y transformación de datos.
- Análisis exploratorio de datasets.
- Tratamiento de variables categóricas y numéricas.
- Identificación de valores nulos o irregulares.
- Visualización de datos.
- Evaluación de modelos mediante métricas de rendimiento.
- Aplicación básica de modelos de machine learning.

## Objetivo del repositorio

Centralizar el material desarrollado durante la asignatura, permitiendo tener una estructura organizada de los laboratorios, talleres y proyectos relacionados con el procesamiento de datos de alto volumen.

Este repositorio funciona como evidencia del proceso de aprendizaje y práctica en el uso de tecnologías para análisis y procesamiento distribuido de datos.
