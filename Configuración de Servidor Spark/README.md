# Introducción a PySpark y HDFS

Este cuaderno es una práctica introductoria al uso de **PySpark** para el procesamiento de datos en un entorno distribuido. Se trabaja con un archivo CSV relacionado con casos de ictus o accidente cerebrovascular, usando Spark para cargar, explorar y transformar los datos.

## ¿Qué se usa en el cuaderno?

- Python
- Apache Spark
- PySpark
- Hadoop HDFS
- SparkSession
- DataFrames de PySpark
- Pandas
- NumPy
- Matplotlib
- Seaborn

## ¿Qué se hace en el cuaderno?

En el cuaderno se realiza un flujo básico de trabajo con datos usando PySpark:

- Configuración de una sesión de Spark.
- Lectura de un archivo CSV.
- Carga del dataset en un DataFrame de PySpark.
- Visualización inicial de los datos.
- Revisión de columnas y estructura del dataset.
- Cambio de nombres de columnas.
- Conversión de tipos de datos.
- Revisión de variables categóricas.
- Identificación de valores nulos o irregulares.
- Análisis básico de frecuencias.
- Preparación inicial de los datos para análisis posteriores.
